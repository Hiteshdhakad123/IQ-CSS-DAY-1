Ques. What is the z-index in CSS?

In CSS Z-index property is used to determine the stack order of elements , Z-index usually works for positioned elememnts when there are more than one element on same postion then which element will be shown first of web page . The Z-index value can be negative value also . The Greater the value is the higher priority will be given to element in that stack order .

Ques. What is the difference between padding and margin?

Margin and Padding are two property in CSS which are part box-model, Padding property defines the space between the content and border of the element. Padding value should not be negative value , while Margin is used create space around elements, outside of any defined borders. Margins values can be negative values.
